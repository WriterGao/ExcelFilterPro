# 图标资源

本目录包含应用程序所需的图标文件。

## 文件结构

- `app.ico` - Windows应用程序图标
- `app.icns` - macOS应用程序图标  
- `app.png` - 通用PNG格式图标

## 图标规格

- **尺寸**: 256x256, 128x128, 64x64, 32x32, 16x16
- **格式**: ICO (Windows), ICNS (macOS), PNG (通用)
- **风格**: 现代扁平化设计

## 使用说明

图标会在以下场景中使用：
- 应用程序窗口标题栏
- 任务栏/Dock显示
- 桌面快捷方式
- 安装包图标
